================================================================================
                    СИСТЕМА УПРАВЛЕНИЯ СКУД - ИНСТРУКЦИЯ
================================================================================

ОПИСАНИЕ
--------
Данная система предназначена для учета приходов сотрудников с использованием
карт доступа и считывателя. Система состоит из двух приложений с графическим
интерфейсом (GUI):

1. admin-app.exe  - Админ-панель для управления сотрудниками
2. check-app.exe  - Приложение для отметки сотрудников


================================================================================
БЫСТРЫЙ СТАРТ
================================================================================

1. Скопируйте все файлы в нужную папку (например, на Рабочий стол)
2. Запустите admin-app.exe для добавления сотрудников
3. Разместите check-app.exe в месте прохода сотрудников
4. Сотрудники прикладывают карту для отметки


================================================================================
ПРИЛОЖЕНИЕ 1: ADMIN-APP.EXE (Админ-панель с GUI)
================================================================================

ИНТЕРФЕЙС:
  ┌─────────────────────────────────────────────────────────┐
  │  СКУД - Админ-панель                                    │
  ├─────────────────────────────────────────────────────────┤
  │  [➕ Добавить]  [🔄 Обновить]              [Выход]      │
  ├─────────────────────────────────────────────────────────┤
  │  Список сотрудников:                                    │
  │  ┌───────────────────────────────────────────────────┐  │
  │  │ Иванов И.И. | Карта: 1234567890        [✏️] [🗑️] │  │
  │  │ Петров П.П. | Карта: 0987654321        [✏️] [🗑️] │  │
  │  │ Сидоров С.С. | Карта: 1122334455       [✏️] [🗑️] │  │
  │  └───────────────────────────────────────────────────┘  │
  └─────────────────────────────────────────────────────────┘

ФУНКЦИИ:

➕ ДОБАВИТЬ СОТРУДНИКА:
  1. Нажмите кнопку "Добавить сотрудника"
  2. В открывшемся окне введите ФИО сотрудника
  3. Приложите карту к считывателю (ID появится автоматически)
     или введите ID вручную
  4. Нажмите "Создать"
  5. Сотрудник будет добавлен в базу данных

✏️ РЕДАКТИРОВАТЬ СОТРУДНИКА:
  1. В списке сотрудников нажмите кнопку [✏️] напротив нужного сотрудника
  2. В открывшемся окне измените ID карты (приложите новую карту)
  3. Нажмите "Обновить"
  4. Карта сотрудника будет обновлена

🗑️ УДАЛИТЬ СОТРУДНИКА:
  1. В списке сотрудников нажмите кнопку [🗑️]
  2. Подтвердите удаление в диалоговом окне
  3. Сотрудник будет удален из базы

🔄 ОБНОВИТЬ СПИСОК:
  - Нажмите кнопку "Обновить список" для перезагрузки данных

ИНТЕРФЕЙС:
  ┌─────────────────────────────────────────────────────────┐
  │  СКУД - Отметка сотрудников                             │
  ├─────────────────────────────────────────────────────────┤
  │                                                         │
  │                    👤 (иконка)                          │
  │                                                         │
  │         Приложите карту к считывателю                   │
  │                                                         │
  │  ┌─────────────────────────────────────────────────┐    │
  │  │ ID карты: [___________________]                 │    │
  │  └─────────────────────────────────────────────────┘    │
  │                                                         │
  │              [✓ Обработать]                             │
  │                                                         │
  │  ✅ Отметка успешна!                                    │
  │  Сотрудник: Иванов Иван Иванович                        │
  │  Время: 08:15:30                                        │
  │  Дата: 30.03.2026                                       │
  │                                                         │
  │  [Выход]                                                │
  └─────────────────────────────────────────────────────────┘

ИСПОЛЬЗОВАНИЕ:
  1. Запустите check-app.exe
  2. Приложите карту к считывателю
  3. ID карты автоматически появится в поле ввода
  4. Программа проверит карту и запишет отметку
  5. Через 3 секунды интерфейс сбросится для следующей отметки

СООБЩЕНИЯ:
  ✅ Отметка успешна!    - Сотрудник успешно отмечен
  ❌ Карта не считана    - Поле пустое, приложите карту
  ❌ Сотрудник не найден - Карта не зарегистрирована в базе


================================================================================
ОТЧЕТЫ В EXCEL
================================================================================

ПАПКА С ОТЧЕТАМИ:
  Отчеты хранятся в папке "Otchet", которая создается автоматически
  в той же директории, где находится check-app.exe

ИМЕНА ФАЙЛОВ:
  Каждый день создается новый файл с именем в формате: ГГГГ-ММ-ДД.xlsx
  Пример: 2026-03-30.xlsx

СТРУКТУРА ТАБЛИЦЫ:
  | Время   | ФИО              | Карточка        |
  |---------|------------------|-----------------|
  | 08:15:30| Иванов И.И.      | 1234567890      |
  | 08:20:15| Петров П.П.      | 0987654321      |

ОСОБЕННОСТИ:
  - Новый файл создается при первой отметке дня
  - Повторные отметки добавляются в существующий файл
  - Все отметки за день хранятся в одном файле


================================================================================
БАЗА ДАННЫХ
================================================================================

ФАЙЛ БАЗЫ ДАННЫХ:
  skudscript.db - файл базы данных SQLite
  Создается автоматически при первом запуске admin-app.exe

ТАБЛИЦА USERS:
  - id        - уникальный номер записи (автогенерируется)
  - card_id   - ID карты доступа (уникальный)
  - full_name - ФИО сотрудника

ПЕРЕНОС ДАННЫХ:
  Для переноса системы на другой компьютер скопируйте:
  - admin-app.exe
  - check-app.exe
  - skudscript.db (если нужно сохранить данные)
  - папку Otchet (если нужно сохранить отчеты)


================================================================================
ТРЕБОВАНИЯ К ОБОРУДОВАНИЮ
================================================================================

ОПЕРАЦИОННАЯ СИСТЕМА:
  - Windows 7 и выше

СЧИТЫВАТЕЛЬ КАРТ:
  - Любой USB считыватель, работающий в режиме эмуляции клавиатуры
  - При считывании карты считыватель автоматически:
    * Вводит ID карты в поле ввода
    * Нажимает Enter

ПРИМЕРЫ СЧИТЫВАТЕЛЕЙ:
  - Z-2 USB
  - SmartCard Reader
  - Любые совместимые HID считыватели


================================================================================
ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
================================================================================

ПРОБЛЕМА: Карта не считывается
РЕШЕНИЕ: 
  - Проверьте подключение считывателя к USB
  - Убедитесь, что считыватель работает в режиме эмуляции клавиатуры
  - Проверьте, что карта не повреждена

ПРОБЛЕМА: "Карта не найдена"
РЕШЕНИЕ:
  - Запустите admin-app.exe
  - Добавьте сотрудника с данной картой

ПРОБЛЕМА: Ошибка при создании отчета
РЕШЕНИЕ:
  - Проверьте права доступа к папке с программой
  - Убедитесь, что на диске есть свободное место

ПРОБЛЕМА: Не отображаются русские символы в GUI
РЕШЕНИЕ:
  - Убедитесь, что в системе установлены шрифты с поддержкой кириллицы
  - Перезапустите приложение

СБОРКА ИЗ ИСХОДНЫХ КОДОВ
================================================================================

ТРЕБОВАНИЯ:
  - Go 1.21 или выше (https://golang.org/dl/)
  - Интернет для загрузки зависимостей

ИНСТРУКЦИЯ ПО СБОРКЕ:
  1. Откройте командную строку в папке проекта
  2. Выполните: go mod tidy
  3. Выполните: build.bat
  4. Готовые файлы появятся в папке dist/

СОЗДАНИЕ АРХИВА:
  1. Выполните: create-archive.bat
  2. Будет создан архив skudscript_release.zip

================================================================================
                    ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
================================================================================

ПРИМЕР 1: ДОБАВЛЕНИЕ НОВОГО СОТРУДНИКА
  1. Запускаем admin-app.exe
  2. Нажимаем "Добавить сотрудника"
  3. Вводим: Иванов Иван Иванович
  4. Прикладываем карту к считывателю
  5. Нажимаем "Создать"
  6. Готово! Сотрудник добавлен

ПРИМЕР 2: ОТМЕТКА ПРИХОДА
  1. Запускаем check-app.exe
  2. Сотрудник прикладывает карту
  3. Программа записывает время и ФИО в Excel
  4. Через 3 секунды готова к следующей отметке

ПРИМЕР 3: ИЗМЕНЕНИЕ КАРТЫ
  1. Запускаем admin-app.exe
  2. Находим сотрудника в списке
  3. Нажимаем [✏️]
  4. Прикладываем новую карту
  5. Нажимаем "Обновить"
  6. Карта обновлена!

ПРИМЕР 4: ПРОСМОТР ОТЧЕТА
  1. Открываем папку Otchet
  2. Находим файл с текущей датой
  3. Открываем файл в Microsoft Excel или LibreOffice Calc
  4. Видим все отметки за день
